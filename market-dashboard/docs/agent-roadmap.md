# 财经 Agent 数据与研究路线图

## 当前完成

- 数据盘迁移：Postgres/Grafana 数据目录已放到 `/www/market-dashboard-data`。
- AKShare 精简：只保留当前服务器上稳定的数据源，包括 A股市场资金流、公司新闻、财务指标。
- Yahoo Finance 日线：新增 A股指数、A股样本、美股/ETF 样本的日线数据。
- 统一日线表：新增 `daily_bars`，保存 OHLCV、成交量、来源、市场和资产类型。
- 动态问答：`/ai/api/ask` 会根据问题识别代码、中文名、英文名和主题别名，查询日线、文本证据、SEC、宏观和数据状态，再调用模型回答。
- 问答留痕：新增 `agent_question_answers`，保存问题、回答、模型和检索上下文。
- 最近问答 API：`/ai/api/qa/recent`。
- SEC 正文证据：24 家核心美股公司 filings 已扩展，最近 4 份 filing 保存清洗正文片段。
- 证据搜索 API：`/ai/api/evidence/search?q=关键词` 可搜索新闻、SEC 正文、财务文本和资产别名。
- A股公告：新增 `cn_company_announcement`，按重点 A股样本同步最近 90 天公告元数据和公告类型。

## 当前数据覆盖

- A股指数日线：上证、深成指、创业板、沪深300、中证500、中证1000、科创50。
- A股样本日线：30 个核心 A股标的，覆盖白酒、银行、保险、新能源、半导体、医药、红利、能源、家电、券商等。
- 美股/ETF 日线：40 个美股/ETF 标的，覆盖大型科技、半导体、金融、能源、医疗、消费、宽基指数 ETF、长债、信用债、黄金和原油。
- 资产别名：`asset_aliases` 已覆盖 296 个中文/英文别名和主题词，例如英伟达、宁德时代、半导体ETF、长债、黄金、白酒、银行等。
- 宏观：FRED 全球市场、利率、信用、外汇、商品、通胀、就业、流动性。
- SEC：24 家核心美股公司 filings 和 company facts；当前约 288 条 filing，其中 96 条带正文片段，152 条 fundamentals series。
- A股公告：当前重点样本约 149 条公告，包括一季报、董事会决议、融资/配售、并购问询回复等。

## 下一步建议

1. 继续扩大资产池
   - A股从 30 个样本扩到 100-300 个核心股票/ETF。
   - 美股从 40 个核心标的扩到 100 个左右的科技、金融、能源、医疗、消费和 ETF。

2. 增强检索
   - 补充更多中文公司名、行业、主题词的资产映射。
   - 增加更细的行业/ETF 归类，例如 AI 应用、机器人、创新药、港股科技、红利低波。

3. 强化证据链
   - SEC 8-K/10-K 正文已经入库，下一步做结构化摘要和 item-level 解析。
   - A股公告元数据已经入库，下一步抓取公告正文/PDF 并做摘要。
   - 每次回答输出固定格式：事实、推断、风险、数据缺口、证据日期。

4. 多 Agent 拆分
   - `MarketDataAgent`：价格、趋势、成交量、波动。
   - `MacroAgent`：利率、美元、信用、商品、流动性。
   - `FundamentalAgent`：财务、估值、SEC facts。
   - `NewsFilingAgent`：公告、新闻和事件。
   - `RiskAgent`：反方观点和风险清单。
   - `ReportAgent`：合成中文研究报告。

5. 质量控制
   - 回答必须显示数据截止日期。
   - 回答必须说明数据缺口。
   - 不给实盘下单、仓位比例、保证收益。
   - 重要结论必须能追溯到数据库证据。
